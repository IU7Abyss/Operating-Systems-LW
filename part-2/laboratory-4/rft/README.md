# Recursive File Tree
## Проверено
- OS X 10.10
- Linux 3.8.0-29

## Собираем
```
$ make
```

## Копируем
#### OS X
```
$ make install
```
#### Linux
```
# make install
```

## Юзаем
```
$ rft [path]
```
По дефолту `path = .`. Если передать больше одного аргумента ничего не произойдёт.

## Зачищаем
#### OS X
```
$ make uninstall
```
#### Linux
```
# make uninstall
```

## Сворачиваем
```
$ make clean
```
